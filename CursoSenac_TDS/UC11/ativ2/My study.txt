Para enviar todas as alterações para o repositório remoto no GitHub, siga estes passos:

### **1. Verifique o status para garantir que tudo foi adicionado**
```bash
git status
```
Se os arquivos ainda aparecerem como "Changes not staged for commit", certifique-se de que rodou:
```bash
git add .
```

---

### **2. Faça o commit das alterações**
Adicione uma mensagem descritiva ao commit:
```bash
git commit -m "Descrição das alterações feitas"
```

---

### **3. Envie as alterações para o repositório remoto**
```bash
git push origin uc11
```
Isso enviará as alterações para o branch remoto **uc11**.

---

### **4. Verifique se o push foi bem-sucedido**
Para confirmar que tudo foi enviado corretamente, use:
```bash
git status
```
Se aparecer **"Your branch is up to date with 'origin/uc11'"**, então está tudo certo!
