Ele será um programa que irá rodar localmente ou será uma extensão de navegador?
Irá rodar localmente. É muito mais simples do que parece, uma explicação rápida já trará um bom entendimento.

Então irei filtrar utilizando palavras chaves e agrupar. Apenas para ajudar na organização das conversas. No ambiente industrial muitas vezes vc quer apenas as mensagens que tal número mandou no grupo. Aí então o usuário seleciona o numero, faço uma filtragem e apresento apenas as mensagens daquele número.




Pensando que as mensagens serão consumidas do whatsapp, como você imagina esse processo ocorrendo?
Elas não serão consumidas no whatsapp. Serão baixadas do aplicativo whatsapp
em um formato característico, ou seja esse processo é externo. Meu projeto receberá a pasta com arquivos padrão do whatsapp. Recebendo estes arquivos 
só vou precisar usar JAVA para manipular as strings para que eu possa organizar e armazenar no banco de dados.
